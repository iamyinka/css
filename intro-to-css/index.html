  • Selector (elements e.g h1, h2, p span, classes, ids e.t.c)
  • Property (ies) (color, background-color, font-size, padding, marging, e.t.c)
  • Value(s) (corresponing values of each property, e.g red, 16px e.t.c)

Example like this 


        selector {
            property: value;
        }
